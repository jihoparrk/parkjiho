신입 웹 퍼블리셔 박지호입니다!


컴퓨터로 자주 쓰는 루틴을 매크로로 만들 수 있지 않을까 싶어 구글에 검색하였던 것이 코딩 입문의 계기였습니다.

프로그래밍에 대해 전혀 몰랐기 때문에 문법 하나 하나가 어떤 의미인지 검색하고 인터넷 강의를 찾아가며 만들었는데, 이 과정이 굉장히 흥미롭게 느껴졌습니다.

업계에 관심이 생겨 어떤 분야가 있는지 찾아보게 되었고, 웹 퍼블리싱에 매력을 느껴 6개월간 학원을 다니며 관련 교육과정을 거쳤습니다

탄탄한 마크업을 기반으로 사용자의 기억에 남는 퍼블리싱을 하고싶습니다 나무처럼 차근차근 성장하여 단단히 뿌리를 내리는 웹 퍼블리셔를 꿈꿉니다

  • CSS3

    웹 호환성을 고려한 크로스 브라우징

  • JS

    JavaScript

    기본적인 자바스크립트에 대한 이해도를
    바탕으로 제이쿼리와 병행하여 사용합니다

  • jQuery

    애니메이션 효과 및 대화형처리에 제이쿼리를
    적용하여 짧은 코드로 동적기능을 구현합니다

  • B

    Bootstrap

    필요 시 부트스트랩을 사용하여
    간단하고 신속한 작업도 가능합니다

  • React

    리액트에 대한 기본적인 이해가
    있으며 계속해서 공부중입니다

  • slick

    Sass & Slick

    Sass나 Slick과 같은 다양한 라이브러리와
    프레임워크를 활용하여 신속하게 작업합니다
